奶油義大利麵

材料：

  • 意大利面：
    • 干意大利面
    • 食盐
  • 奶油酱：
    • 黄油
    • 奶油
    • 大蒜（切碎）
    • 奶酪（帕玛森芝士或其他喜好的奶酪）
    • 盐和胡椒粉
  • 其他配料（可选）：
    • 新鲜香草
    • 火腿丁
    • 蔬菜（例如菠菜、樱桃番茄等）

制作步骤：

  1. 步骤1：煮意大利面

    2. 在盐水中煮熟意大利面，然后沥干。

  2. 步骤2：制作奶油酱

    3. 在锅中融化黄油，加入切碎的大蒜，然后倒入奶油和奶酪，搅拌至奶油和奶酪融化，调味盐和胡椒粉。

  3. 步骤3：拌炒意大利面

    4. 将煮熟的意大利面加入奶油酱中，充分拌炒，确保面条均匀裹上奶油酱。

  4. 步骤4：加入其他配料（可选）

    5. 根据个人口味，可以加入新鲜香草、火腿丁、蔬菜等其他配料。

典故

在中文中，義大利麵顧名思義指來自義大利的麵食。
在義大利語中，Pasta一詞來自拉丁語，其在拉丁語中義為「麵團」。

其最初的來源則是希臘語（古希臘語：παστά，羅馬化：pastá，直譯：大麥粥）。 Pasta一詞 首次出現在英語中是1874年。
